输入输出运算

Created2022-12-21|Updated2022-12-29
|Word count:820|Reading time:3min|Post View:

输出

特殊字符的输出

转义字符：” \ “后面的字符不代表本身的含义

字符 含义
\n 换行
\f 换页
\r 回车
\ \ 表示一个反斜杠字符 ‘ \ ‘
\ ‘ 表示一个单引号字符 ‘ ‘ ‘
\ ‘’ 表示一个双引号字符 ‘ “ ‘
? 表示一个问号字符 ‘?’
\0 空字符（NULL）

常用的输出控制符

控制符 含义
%d 按十进制整型数据的实际长度输出
%ld 输出长整型数据
%md m为指定的输出字段的宽度
%u 输出无符号整型
%c 输出一个字符
%f或%lf 输出实数，包括单精度和双精度，以小数形式输出
%.mf 输出实数时小数点后保留m位
%s 输出字符串

输入与记忆

变量的数据类型

数据类型 类型标识符 存储空间大小(1字节=8bits) 备注
字符型 char 1 不可用于运算，主要用于输入输出
短整型 short 2 可运算
整型 int 4 可运算
长整型 long 4 可运算
单精度型 float 4 实型，可运算
双精度型 double 8 实型，可运算
布尔型 bool 1 只有真(true)和假(false)，用于逻辑运算

变量的赋值

语句 是否符合语法规则
a=b=c=d=e=5; 是
int a=5,b,c; 是
int a=b=c=5; 否
int a=5,b=6,c;c=a+b; 是

运算

“++”运算符 ( ‘’–’’ 同 )

表示自加一，相当于i=i+1。

“++”运算符有 “i++” 和 “++i” 两种形式，前者表示先赋值在计算，后者表示先计算再赋值。

关系表达式与关系运算

优先级： 算术运算符>关系运算符>赋值运算符

​ a+b<c+d 等效于 (a+b)<(c+d)

​ a==b<c 等效于 a==(b<c)

​ (a=3)!=(b=3) 输出0 注意区分关系运算符“==”和赋值运算符“=”

运算结果：是一个逻辑值，即“真”或“假”

逻辑表达式与逻辑运算

推荐使用小括号()，以便更明确地表示运算优先级

优先级：逻辑非>算术运算符>关系运算符>逻辑与>逻辑或>赋值运算符

运算结果：是一个逻辑值，即“真”或“假”（结果为“0”则为“假”，结果“非0值”，则为“真”）
